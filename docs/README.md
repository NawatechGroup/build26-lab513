# Lab Documentation

This folder contains the hands-on lab content for **LAB513: Build an AI app with Azure SQL Hyperscale, Microsoft Fabric, and Microsoft Foundry**.

Use these docs to work through the lab from prerequisites to implementation. The exercises cover Azure SQL vector search, GitHub Copilot-assisted SQL development, Retrieval-Augmented Generation, Microsoft Foundry Agents, Fabric analytics, and MCP-based access to lab data.
